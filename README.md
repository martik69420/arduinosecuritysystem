# Arduino smart home security system

A C++, on arduino IDE made security that includes,
* A temperature sensor that, when it measures above its limit (400), turns on a buzzer, which only turns off when it goes under the limit.
* A light sensor that, when it measures above its limit (700), turns on an LED, which only turns off when it goes under the limit.
* A crash sensor that kind of works like a button; it has to be pressed, but it's kind of when it gets pushed. When it is pressed, it turns on the buzzer, which then can only be deactivated when the touch sensor is pressed
* The Tilt sensor is very similar in purpose to the crash sensor, when tilted it turns on the buzzer, which again can only be deactivated using the touch sensor.
